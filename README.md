## Project Description
Traditional multi-page church website built with pure HTML, CSS, and JavaScript for St. Francis of Assisi Anglican Church in Mamelodi West.

## Features
- **7-Page Structure**: Home, About, Ministries, Services, Donations, Contact, Structure
- **Gmail-Only Validation**: Contact forms accept only @gmail.com addresses
- **South African Focus**: Phone validation for SA numbers only
- **iOS Enhancements**: Special visual effects for iOS 26+ users
- **Cookie Compliance**: Banner with 6-month consent memory
- **Security Layer**: Client-side protection with VPN detection
- **Heavenly Design**: Serene, sanctuary-inspired aesthetics

## Technical Details
- **Built With**: HTML5, CSS3 (CSS Modules/BEM), Vanilla JavaScript
- **No Frameworks**: No React, Tailwind, or external CSS frameworks
- **Responsive**: Mobile-first design
- **External Services**: Google Maps embed, free geolocation API

## Color Scheme
- **Primary Blue**: `#003366` (trust, Anglican heritage)
- **Accent Gold**: `#D4AF37` (warmth, spirituality)
- **Background**: White and `#f9f9f9` (clean, readable)

## Fonts
- **Headings**: Elegant serif (Playfair Display/Cormorant Garamond)
- **Body**: Clean sans-serif (Inter/Open Sans)
- **Accents**: Decorative script (Dancing Script) for special elements

## Special Features
- **iOS Gloss Effect**: Exclusive styling for iOS 26+ devices
- **Ministry Grid**: 12 interactive cards for church ministries
- **Smart Cookie Banner**: Remembers consent for 6 months
- **Security Persona**: "Griffin" themed client-side protection
- **Form Protection**: Prevents iOS zoom and validates inputs
